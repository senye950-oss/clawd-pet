Clawd

A tiny pixel art desktop pet for your web page. Zero dependencies, ~5KB.

State

Actions

Try it

Poke it. Drag it. Push it to the edge. Poke it 3+ times fast. Try at night (after 23:00). Long-press does nothing in the demo (in production, it opens the line editor).

// 1. Include the script
<script src="clawd.js"></script>

// 2. Create and mount
var pet = new Clawd({
  lines: {
    poke: ['hi', 'boop', 'hey!'],
    rare: ['secret!']
  }
}).mount()

// 3. Control state from your app
pet.busy() // AI is responding
pet.idle() // AI finished
pet.offline() // disconnected
